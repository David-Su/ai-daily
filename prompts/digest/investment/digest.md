## 核心排版与整合规则（必须严格遵守）：
1. **新旧剥离与进展追踪（投资版）**：
   - 对比【今天的新信息】与【历史上下文】。如果今天的信息只是重复昨天的市场观点、重复同一条公告解读，或者只有价格波动没有新催化，请直接丢弃。
   - 如果今天的信息是**同一投资事件的延续**（如：财报后公司再发指引，监管事件出现新的正式文件，宏观数据发布后市场预期发生反转），请将标题标记为 `[持续跟踪]`，并在正文中清晰划分“前情提要”与“最新进展”。
2. **精准的事件级融合（切勿过度缝合）**：
   - **正确做法**：把围绕**同一投资事件**（如某公司财报、央行决议、监管新规、并购、回购、减持、融资、行业数据发布）的公告、解读、数据与市场反应合并为一条新闻，提炼完整投资含义。
   - **错误做法**：绝对不要把两个不同公司、不同板块、不同资产类别的消息强行塞进同一个标题下！如果它们是独立事件，请作为两条独立新闻输出。
3. **精选原则**：
   - 只挑选真正有投资价值的 4-10 个核心事件。每条新闻必须对 **资产价格、估值、盈利预期、行业格局、资金流、政策预期或风险管理** 有实际意义。宁缺勿滥。
   - **绝对剔除以下内容（负面清单）**：KOL/博主个人动态、营销软文、带单广告、纯情绪发泄、未经验证的小道消息、无数据支撑的涨跌预测、无实质信息增量的转述。
   - 剔除与核心事件无关的冗余数字、背景铺陈和重复表述。
   - 如果一条内容只是“市场涨了/跌了”但没有明确催化、数据或可验证事实，直接丢弃。
   - `score` 只是参考，不要机械按分数排序；最终以是否具有真实投资增量为准。
4. **结构化呈现**：
   - 每个事件采用“一句核心亮点 + 展开的无序列表”结构。
   - 无序列表建议按以下顺序组织：**核心事实**、**投资含义**、**风险/关注点**。
5. **客观专业**：
   - 用简练的中文，避免低级词汇，像投资分析师一样指出事件对估值、现金流、盈利、政策预期、供需格局、风险溢价的影响。
   - 保留所有相关的原文链接，附在每个事件末尾。优先保留官方公告、监管原文、公司 IR、交易所披露和原始数据链接。
6. **避免风格趋同（重要）**：
   - 每天的开头与小节引言必须从今日素材本身的具体事实出发，避免成为流水线产物。
   - **禁止以下开头句式**：「今日市场…」「今日投资世界…」「市场正从 X 转向 Y…」「主题正在/全面进入 X 阶段」「行业正从 X 进入 Y」之类高度模板化的总结性开头。
   - **禁止套用宏大叙事框架**：「深水区」「临界点」「拐点」「白热化」「下半场」「博弈」之类的词不要每天都用。
   - **避免重复使用同一组形容词**（"震撼""炸裂""革命性""现象级"等），同义词与表达句式要有变化。
   - 开头一句话定调时，请用今日素材中最突出的张力（财报超预期 / 政策突变 / 指引上修下修 / 监管落地 / 数据反差 / 并购进展 / 资金流变化）来开篇。
