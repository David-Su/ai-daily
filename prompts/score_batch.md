## 任务与评分标准
根据所述领域为每条信息打分（0-100）：
### AI
- 【90-100分】首发重大新闻、里程碑级发布。
*例子：Google官方宣布Nano Banana 2发布；OpenAI官方宣布新模型；Anthropic拒绝美国防部合同（首发报道）。*
- 【80-89分】重要技术进展、知名大佬的核心观点、深度长文。
*例子：某知名开发者开源了一个能让性能提升50%的框架；Jeff Dean对某项新技术的深度解析长推文。*
- 【70-79分】实用工具更新、有价值的教程、行业数据报告。
*例子：Cursor新增了某个小功能；某博主分享的Claude Code 45个实用技巧。*
- 【60-69分】二手信息、一般性新闻、小道消息。
*例子：某媒体转发了Google发布的推文（非首发）；未经验证的传言。*
- 【<60分】低价值/局部内容：纯情绪宣泄、无营养的评价、广告、日常闲聊、KOL个人生活动态、硬广软文、无营养的二手评价。
*例子：“Nano Banana 2太牛了！”（无具体测试数据）；“Anthropic真硬气”（纯情绪）；“快来买我的课”。*
### 投资
- 【90-100分】可能显著影响资产价格或投资判断的一手重大信息、官方公告、监管政策、宏观数据、公司核心事件。
*例子：上市公司正式发布超预期财报或重大业绩预警；美联储/央行公布关键利率决定；SEC/证监会发布重大监管规则；公司宣布重大并购、破产、退市、重大回购、分拆或核心管理层变动；权威机构发布影响市场预期的通胀、就业、GDP数据。*
- 【80-89分】高质量投资分析、重要行业变化、机构核心观点、深度基本面研究、估值和风险框架清晰的长文。
*例子：知名机构发布对某行业盈利周期的深度报告；资深投资人系统解释一家公司护城河、估值和风险；对财报、现金流、资产负债表或竞争格局有扎实拆解的研究文章。*
- 【70-79分】有实用价值的市场数据、投资工具、财报跟踪、行业报告、策略复盘和风险提示。
*例子：ETF资金流向统计；某行业月度销量或库存数据；财报日历和重点指标整理；宏观数据图表；组合管理、仓位控制或估值方法教程。*
- 【60-69分】二手信息、一般性市场新闻、简短观点、未经充分验证但可能需要关注的传闻。
*例子：媒体转述公司公告但缺少增量信息；KOL简短评论市场涨跌原因；未证实的并购、融资、政策或业绩传闻。*
- 【<60分】低价值或高噪声内容：情绪化喊单、无数据支撑的涨跌预测、营销软文、带单广告、日常闲聊、纯观点宣泄、标题党、重复搬运、缺少事实依据的阴谋论。
*例子：“这只股票必涨十倍”（无数据和逻辑）；“今天市场太吓人了”（纯情绪）；“加入群领取牛股”；只复制股价涨跌但没有解释和增量信息。*
### 其他
0分

## 输出要求
必须返回纯 JSON 数组，每个对象包含：
- `link`: 原文链接（必须保留原样）
- `domain`: 所属领域（严格按照所属领域归类标准进行归类）
- `score`: 整数评分（严格按照上述标准，宁缺毋滥）
- `tags`: 标签（如 ["大模型", "开源", "教程"]），事件聚类词（最多5个字，用于把讨论同一件事的推文归类，例如 "Nano Banana 2发布", "DoD与Anthropic", "Cursor更新"）
- `summary`: 一句话客观摘要（提取核心事实，去掉主观情绪，50字内）

}
